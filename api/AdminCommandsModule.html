AdminCommandsModule (or CommandClass or CommandClassV2) was made mainly for Union Developments Admin Commands, powered by OOP (Object-Oriented Programming)

Properties

string CommandClass.Prefix
The command prefix that will be used by start()
 
Array CommandClass.Commands
A list of Commands that will be used by the runner.
 
Array CommandClass.Permission
A list of Permissions that will be used by the commands.
 
Dictionary CommandClass.Utilities
A list of functions that can be used by anyone.
(Mostly useful while using it with the module.)
 
Enums CommandClass.Enum
A list of Enums that can be used by anyone.
(Mainly for the module itself.)

Functions

Command CommandClass.new(string Name, function Function, array SpecificPermission [optional])
Returns a new Command Class and list it into the CommandClass.Commands array.

A function that was used by the module only.
